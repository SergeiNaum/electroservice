##### Download image from docker hub

---
```

```
---

##### create directory app and navigate to it

```
mkdir app && cd app
```
---

##### clone project 

```
git clone https://github.com/SergeiNaum/electroservice.git
```
---

##### Start app execute the command
```
docker-compose up
```
